# Changelog

## 0.1.0-alpha.0

Initial public alpha target.

- Adds the `lattice` CLI for local graph build/search/impact, edit planning, checks, and pre-write validation.
- Ships public packages for CLI, contracts, graph, edit, validation, and TypeScript validation.
- Keeps fixtures internal for tests and examples.
- Supports native graph artifacts on `darwin-arm64`, `darwin-x64`, `linux-x64`, and `linux-arm64`.
- Documents Windows as unsupported for this alpha.
- Adds release dry-run and maintainer publish workflow scaffolding.
